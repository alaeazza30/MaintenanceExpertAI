# 🔧 MaintenanceExpertAI

AI-powered Industrial Maintenance Decision Support System.

## Features

- Equipment analysis
- FMECA/AMDEC generation
- RPN calculation
- Spare parts optimization
- Maintenance action planning
- Google Gemini integration
- Streamlit dashboard

## Technologies

- Python
- Google Gemini
- Google ADK
- Streamlit
- Git/GitHub

## Project Architecture

```
MaintenanceExpertAI
        ↓
Equipment Analysis
        ↓
FMECA Generation
        ↓
RPN Calculation
        ↓
Spare Parts Optimization
        ↓
Maintenance Action Plan
```
